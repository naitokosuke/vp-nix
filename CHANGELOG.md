# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## Versioning Scheme

This project uses the format: `<vite-plus-version>-nix.<revision>`

- When vite-plus upstream updates: Update the vite-plus version part
- When vp-nix changes: Increment the `-nix.X` revision
